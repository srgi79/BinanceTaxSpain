# BinanceTaxSpain
Calculate Profits and Tax using Binance reports in CSV and API

# Usage
- Get CSV reports from Binance web every 3 mounths max
- Create a read-only API acces from Binance web

# Funcionality
This setup will merge all the CSV files in one per year, then it will create a csv formatted to iterate with binance client.
The CSV exported can be used to get your portfolio value and compute the taxes
